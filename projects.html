Website Development

Trulia Recreated

Recreated a Trulia search results page using a combination of flexbox and CSS grid to create uniform positioning of images/results and media queries for mobile responsiveness.

Airbnb Recreated

Recreated the Airbnb homepage utilizing flexbox and the Chrome developer tools to mirror the styles and fonts used per element accurately. I used the ":before" selector and position absolute/relative properties to insert a masked overlay to improve the readability of written content within containers with images as the background.

CCE Landing Page

Created a landing page for my uncle's catering business using a combination of HTML5, Bootstrap 4, CCS3, Javascript and a script created in Gmail to send form responses.

  • Bootstrap elements for the navigation bar and hamburger to create a drop-down menu when viewing on mobile.
  • Bootstrap for the age model that, using JQuery, will appear after 500 milliseconds of the site loading.
  • The age model uses Javascript to calculate the user's age to access the website once the user enters their birthdate and verifies that they are older than 21, an error message will display if a birthdate that does not validate to being 21 or older.
  • Additionally, a form at the bottom of the page uses a script created within GMail to collect responses in a Google Sheet and send an email via a Gmail account.
